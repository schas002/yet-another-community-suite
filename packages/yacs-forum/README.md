# yacs-forum

> Yet another forum

## Install

The forum can't be installed as a normal NPM package. Install dependencies:

```sh
$ yarn install
```

## Usage

```sh
$ yarn start
```

Boots the app running on whatever port you have `$PORT` set to, or 3000 by default.
